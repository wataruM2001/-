# マーチャオ三麻スコア卓

添付 Google Docs / Google Sheets のルールを元にした、3人麻雀の半荘スコア管理アプリです。

## 対応範囲

- 3人打ち、112牌構成の表示
- 半荘開始時の東家・南家・西家をランダム決定
- 東南戦として東1局、東2局、東3局、南1局、南2局、南3局の順に進行
- 親は常に東家、次に親が回る子を南家、その次を西家として表示
- 親アガリ、または流局時の親聴牌なら親継続
- 子アガリ、または流局時の親ノーテンなら南家へ親移動
- 本場は0から開始し、親アガリまたは流局で+1、子アガリで0にリセット
- 南3局はオーラス。親が単独1位なら強制終了し、単独1位でない親アガリ/親聴牌流局では続行か終了を選択
- 同点順位は東1局開始時の東家、南家、西家の順で決定
- リーチ成立時に1000点を供託へ移動し、宣言牌で放銃したリーチ不成立はボタンを押さない運用
- アガリ者が供託を全回収、流局時は次局へ持ち越し、ダブロン時は上家取り
- 半荘終了時に供託が残っていればトップ者が回収
- 流局時は聴牌者チェックからノーテン罰符2000点を精算
- 祝儀は点棒とは別に、1枚/1役/裏ドラ1翻あたり500ポイントで支払い元と受け取り先を記録
- ツモ祝儀は他家2人が同額支払い、ロン祝儀は放銃者のみ、ダブロン祝儀は放銃者が各アガリ者へ支払い
- 子/親、ロン/ツモ、1〜59ハン、本場加算
- 青5索、青5筒、青華、一発、裏ドラによる祝儀ポイント
- 半荘終了時の順位ポイント、祝儀ポイント、トビ賞ポイント集計
- 局ログ、リセット、直前局の取り消し
- ブラウザの `localStorage` による自動保存
